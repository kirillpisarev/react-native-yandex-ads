# react-native-yandex-ads

ReactNative port for Yandex Mobile Ads SDK

## Installation

```sh
npm install react-native-yandex-ads
```

## Usage

```js
import YandexAds from "react-native-yandex-ads";

// ...

const result = await YandexAds.multiply(3, 7);
```
